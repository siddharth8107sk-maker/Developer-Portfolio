## Getting Started

### Clone the repository

```bash
git clone "https://github.com/siddharth8107sk-maker/Developer-Portfolio.git"
```

### Navigate into the project

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
